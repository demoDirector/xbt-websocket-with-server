Demo I: Bitcoin Price in Real Time—an Implementation of the HTML 5 WebSocket Interface
Developing the Demo
I programmed the Web page, and the JavaScript client (client) using the following tools and programming languages: The BitFinex WebSocket API, NetBeans integrated development environment (IDE) 8.2, Cascading Style Sheets, level 2 (CSS2), JQuery, JavaScript Object Notation (JSON), Highcharts, and HTML 5, which includes the WebSocket interface. I developed this software on a computer running Microsoft Windows 8.1. For debugging the client, I used the Google Chrome DevTools Console.

A Demo with a WebSocket!

The programming demo consists of the Web page you are viewing and the JavaScript client that informs the Web page. I programmed the client to create a WebSocket connected to BitFinex, a bitcoin exchange that offers real-time bitcoin price data to subscribers.

The WebSocket enables simultaneous communication between the BitFinex WebSocket server and the JavaScript client. A communication received at a WebSocket is represented programmatically as an event object that contains a message. The box labeled Activity at the WebSocket, in the upper left of the Web page, indicates incoming and outgoing events at the WebSocket of the JavaScript client.
